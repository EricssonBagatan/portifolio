  <div class="lesson">
    <h2>Lesson 1 & 2: Introduction to CSS</h2>
    <p>
      CSS, or Cascading Style Sheets, is used to style and lay out web pages, including changing fonts, colors, and spacing. CSS separates the content of a webpage (HTML) from its design.
    </p>
    <h3>The Three Layers of a Webpage:</h3>
    <ul>
      <li><b>Content Layer:</b> Defines the structure and content of the webpage, written in HTML.</li>
      <li><b>Presentation Layer:</b> Manages the visual styling of the webpage, handled by CSS.</li>
      <li><b>Behavior Layer:</b> Adds interactivity and dynamic behavior, controlled by JavaScript.</li>
    </ul>
  </div>

  <!-- Lesson 3 -->
  <div class="lesson">
    <h2>Lesson 3: Types of CSS Stylesheets</h2>
    <ul>
      <li><b>External Stylesheet:</b> CSS is stored in a separate file (e.g., `styles.css`) and linked to the HTML document.</li>
      <li><b>Internal Stylesheet:</b> CSS is written within a `style` tag in the `head` section of the HTML file.</li>
      <li><b>Inline Stylesheet:</b> CSS is applied directly to an HTML element using the `style` attribute.</li>
    </ul>
  </div>

  <!-- Lesson 4 & 5 -->
  <div class="lesson">
    <h2>Lesson 4 & 5: CSS Selectors, <code>div</code>, <code>span</code>, and Links</h2>
    <ul>
      <li><b>Selectors:</b> CSS selectors target HTML elements for styling (e.g., `h1`, `.class`, or `#id`).</li>
      <li><b>div:</b> A block-level element used to group content for styling or layout.</li>
      <li><b>span:</b> An inline element used to style a specific part of text.</li>
      <li><b>Link:</b> An anchor (`<a>`) element that creates hyperlinks to other pages or sections.</li>
    </ul>
  </div>

  <!-- Lesson 6 -->
  <div class="lesson">
    <h2>Lesson 6: Formatting Text with CSS</h2>
    <p>
      CSS can format text by changing its font, size, weight, style, alignment, and more. For example:
    </p>
    <ul>
      <li><b>Font-family:</b> Changes the typeface (e.g., Arial, Times New Roman).</li>
      <li><b>Font-size:</b> Adjusts the size of the text.</li>
      <li><b>Text-align:</b> Aligns text to the left, right, center, or justify.</li>
      <li><b>Color:</b> Changes the text color.</li>
    </ul>
  </div>

  <!-- Lesson 7 -->
  <div class="lesson">
    <h2>Lesson 7: Formatting Layout with CSS</h2>
    <p>
      CSS provides tools to create page layouts, including:
    </p>
    <ul>
      <li><b>Flexbox:</b> A layout model for creating flexible, responsive designs.</li>
      <li><b>Grid:</b> A two-dimensional layout system for designing complex structures.</li>
      <li><b>Positioning:</b> Controls the placement of elements using `static`, `relative`, `absolute`, or `fixed` positioning.</li>
      <li><b>Margins and Padding:</b> Define spacing around and within elements.</li>
    </ul>
